QBittorrent использует синтаксис **.NET/PCRE** с особыми правилами для групп захвата. Вот **краткая шпаргалка** (cheatsheet) специально для QBittorrent:

## 🎯 **СИНТАКСИС QBITTORRENT REGEX RENAME**

### **1. ГРУППЫ ЗАХВАТА**
```
(?<name>...)  Именованная группа → $name
(группа1)      Нумерованная группа → $1, $2, $3...
```
*Пример из вашего примера:* `(?<qt>qt_)\d+` → группа `qt` = `"qt_"`

### **2. ОСНОВНЫЕ СИМВОЛЫ**
```
\d+    Одна или больше цифр (1, 23, 456)
\d{2}  Ровно две цифры (01, 23, 99)
.*     Любые символы (0 или больше)
.+     Любые символы (1 или больше)
\.     Точка (экранирование!)
^      Начало строки
$      Конец строки
(...)  Группировка
[...]  Класс символов (например [a-z])
```

### **3. ПОДСТАНОВКА**
```
$name      Значение именованной группы
$1, $2...  Значение нумерованных групп
$+        Последняя захваченная группа
$&        Вся совпавшая строка
$`        Текст до совпадения
$'        Текст после совпадения
```

### **4. ВАШ ПРИМЕР С СЕРИАЛАМИ**
Для файлов `01.Название серии.mkv`:

**Search files:**
```regex
^(?<ep>\d{2})\.(?<title>.+)\.(?<ext>mkv|avi|mp4)$
```

**Replacement Input:**
```
$title.s01.$ep.$ext
```

**Результат:** `Название серии.s01.01.mkv`

### **5. ЕЩЁ ПРИМЕРЫ**

#### **Пример 1: Добавление ведущих нулей**
*Исходно:* `ep1.mp4`, `ep12.mp4`
```regex
^(?<prefix>ep)(?<num>\d+)\.(?<ext>mp4)$
```
```replacement
$prefix$num$+$+.$ext
```
*Результат:* `ep001.mp4`, `ep012.mp4` (три цифры всегда)

#### **Пример 2: Перестановка частей**
*Исходно:* `2024-Фильм-1080p.mkv`
```regex
^(?<year>\d{4})-(?<name>.+)-(?<res>.+)\.(?<ext>mkv)$
```
```replacement
$name ($year) $res.$ext
```
*Результат:* `Фильм (2024) 1080p.mkv`

#### **Пример 3: Удаление части**
*Исходно:* `[Anime] Название [1080p].mkv`
```regex
^\[.+?\]\s*(?<title>.+?)\s*\[.+?\]\.(?<ext>mkv)$
```
```replacement
$title.$ext
```
*Результат:* `Название.mkv`

### **6. ОСОБЕННОСТИ QBITTORRENT**
1. **Регистр букв сохраняется**
2. **Тестируйте** в поле поиска torrent'а, набирая regex
3. **Спецсимволы** `$` в Replacement нужно экранировать: `$$`
4. Если не работает, попробуйте оба варианта: `$1` и `$+`

Пример:
- если в regex мы не используем расширение, тогда нужно выбрать **Filename** из выпадающего списка
- обязательно включить галку **Include files** - иначе не заработает!!!
- Для переименования нужно **обязательно** нажать **Replace All** - иначе заменит только один файл, и следующая замена может поломать результат!!!
